# Linux-ollama-
两行命令离线安装 Ollama，造福新手～

使用说明：

1.安装linux环境

2.下载本项目内的install_ollama.sh

3.从https://github.com/ollama/ollama/releases中下载ollama-linux-amd64.tar.zst

4.创建目录（文件夹）

5.将install_ollama.sh和ollama-linux-amd64.tar.zst放在同一目录（文件夹下）

6.右键文件夹界面，点Open Terminal Here（或输入命令把目录切换到install_ollama.sh所在的目录）

7.在命令行输入chmod +x (拖拽install_ollama.sh到这里)，或输入chmod +x ./install_ollama.sh给予执行权限

8.继续输入sudo (拖拽install_ollama.sh到这里)，或输入sudo ./install_ollama.sh运行脚本

9.等待脚本安装

10.在命令行中输入ollama，验证是否成功部署
